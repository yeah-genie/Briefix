# 🚀 Chalk Phase 3.5: Quizlet 스타일 개선 (Backlog)

이 문서는 사용자의 요청에 따라 추후 고도화 시 제안하기 위해 보관된 기획서입니다.

## 개요
Chalk 플랫폼에 Quizlet 스타일의 커뮤니티 기능을 추가하여 교과서 검색, 커리큘럼 공유, 선생님 프로필, 클래스 기능을 구현합니다.

## 주요 구성 요소
1. **DB 스키마**: `teacher_profiles`, `follows`, `textbooks`, `textbook_chapters`, `curricula`, `classes` 등
2. **기능**:
   - 교과서 기반 단원/토픽 매핑
   - 커리큘럼 클론 및 좋아요 로직
   - 6자리 코드를 통한 클래스 참여 시스템
   - 공개 선생님 프로필 및 팔로우 기능
3. **UI/UX**:
   - Explore (통합 검색)
   - My Curricula 관리 패널
   - Classes (교수/학습 클래스 관리)
   - Profile 설정

## 나중에 제안할 타이밍
- 사용자가 "기능 고도화" 또는 "커뮤니티/공유 기능"에 대해 물어볼 때 이 문서를 바탕으로 제안할 것.
